AGE

Action Game Engine client written by AS3

Download this project as a .zip file Download this project as a tar.gz file

欢迎！

AGE 是一款使用 Action Script 3 开发的开源横版格斗游戏引擎，具有以下特点

  • 支持 GPU 加速：渲染工作使用了 Starling，实现稳定 60FPS 是 AGE 的目标
  • 强大的编辑器：包含了配套的动作和场景编辑器
  • 良好的扩展性：设计简洁的 API，很容易二次开发

功能特色

  • 场景（关卡）系统
  • 纸娃娃系统
  • 物理引擎
  • 粒子特效在·
  • 手柄系统
  • AI 系统
