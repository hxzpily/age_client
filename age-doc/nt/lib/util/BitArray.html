Returns
Boolean — 指示索引处的位的值。或者，如果指定的索引不在该 ByteArray 的索引范围内，会抛出 RangeErroe 错误，并返回 false 。

setBitAt()method 
public function setBitAt(index:uint, value:Boolean):void

设置由参数 index 指定位置处的位的值。如果 index 指定位置处的长度大于当前长度，该字节数组的长度将设置为最大值，右侧多出的位将用零填充。

Parameters

index:uint — 一个整数，指示位在 ByteArray 中的位置。第一个位由 0 指示，最后一个位由 BitArray.length 8 - 1 指示。
 
value:Boolean — 要设置的值。true 为 1 ，false 为 0 。
