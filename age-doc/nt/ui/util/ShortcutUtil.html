register()method 
public static function register(combo:Array, callback:Function, scopes:Array = null):void

注册一个组合键 尽管一个组合键可以注册多个 callback，但为每次只有一个组合键生效，并且无法保证顺序 

ShortcutUtil.register([ Keyboard.CONTROL, Keyboard.Z ], undo);

Parameters

combo:Array — 组合键数组，不区分顺序
 
callback:Function — 回调方法
 
scopes:Array (default = null) — 设定作用域，当按键触发时，将根据顺序逐个与 stage.focus 比较（== 或 is），然后调用 callback。

register2()method 
public static function register2(combo:String, callback:Function, scopes:Array = null):void

和 register 一样的功能，但是 combo 参数可以是字符串 

ShortcutUtil.register2("CONTROL+Z", function():void
{
    trace("你按了 CONTROL+Z");
});

Parameters

combo:String — 组合键，以 “+” 分割
 
callback:Function — 回调方法
 
scopes:Array (default = null) — 设定作用域，当按键触发时，将根据顺序逐个与 stage.focus 比较
