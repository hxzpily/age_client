Packagent.assets.extensions
Classpublic class ProgressiveImageAsset
InheritanceProgressiveImageAsset Asset AbstractAsset Object
Implements IAssetUser
Subclasses TextureAsset

渐进式的下载图片 和一般图片不同的是，他会先下载带有 _thumb 后缀的小图，然后再下载大图 小图加载完成后视为加载完成，提示 onAssetComplete() 和 onBitmapDataChange() 大图加载完成后，提示 onBitmapDataChange()
