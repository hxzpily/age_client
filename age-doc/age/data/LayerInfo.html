Packageage.data
Classpublic class LayerInfo
InheritanceLayerInfo Object
Implements IAnimatable

LayerInfo 是场景图层信息 一个场景可以包含任意数目的图层 图层具有下列特性
  • 分背景和对象图层两种
  • 实现了 IAnimatable，当 type 为 LayerType.OBJECT 时，将参与物理计算
