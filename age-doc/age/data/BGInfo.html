Classpublic class BGInfo
InheritanceBGInfo Object

BGInfo 是图层上的背景图，不参与碰撞检测 只允许出现在 LayerInfo.type == LayerType.OBJECT 的图层上 与 ObjectInfo 不同，BG 并不支持 z 排序 也就是说，BG 依赖于所在层进行前后层次的排序 如果强制设置在对象图层上，由于 BG 的 z 视为 0，所以会遮挡后面所有对象
